Materi Dasar HTML

  • Struktur HTML
    Struktur dasar HTML terdiri dari elemen seperti <html>, <head>, dan <body> yang digunakan untuk menyusun kerangka halaman web.
  • Heading dan Paragraf
    Heading (<h1> sampai <h6>) digunakan untuk judul, sedangkan paragraf (<p>) digunakan untuk menampilkan teks atau isi konten.
  • Link
    Link (<a>) digunakan untuk menghubungkan halaman satu dengan halaman lain atau ke website lain.
  • List
    List digunakan untuk menampilkan data dalam bentuk daftar, baik berurutan (<ol>) maupun tidak berurutan (<ul>).
